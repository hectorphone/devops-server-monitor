# DevOps Server Monitor

Repositorio de prácticas de Python y automatización orientadas al aprendizaje de DevOps.

## Estructura
- `automatizacion/`: scripts de automatización y pruebas de archivos.
- `gestor_tareas/`: gestor de tareas básico en Python con manejo de archivos.
- `monitor_servidores/`: script inicial para monitorización de servidores.

## Objetivo
Construir pequeños proyectos prácticos para aprender:
- Python
- Git y GitHub
- automatización
- estructura de proyectos
